# System Design Notes — ContentPilot

Personal system design learning notes, built around ContentPilot.

## How to use
1. Run `python3 ../setup_obsidian_vault.py` to set up your vault
2. Copy these notes into your vault's matching folders
3. Open in Obsidian — diagrams and links work automatically
